¿Qué es el DOM?

El Document Object Model es una interfaz que representa la estructura HTML como un árbol de nodos, permitiendo manipular elementos con JavaScript.

¿Por qué es útil?

Permite crear páginas web dinámicas e interactivas sin necesidad de recargar la página, mejorando la experiencia del usuario.

Métodos principales

createElement(), appendChild(), getElementById(), querySelector(), innerHTML, textContent, y muchos más para manipular el DOM.
