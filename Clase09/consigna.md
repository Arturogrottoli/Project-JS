Bienvenido a la Evaluación Final del Proyecto
Has recorrido un camino sólido en el aprendizaje de JavaScript y desarrollo frontend, desde la configuración del entorno hasta conceptos avanzados como asincronismo y manipulación del DOM. Esta evaluación final no entregable es tu oportunidad para demostrar cómo integras y aplicas estos conocimientos en situaciones prácticas y reales.

Repaso Breve de Conceptos Clave
Antes de sumergirnos en la evaluación práctica, recordemos algunos conceptos fundamentales que serán esenciales:

Variables y tipos de datos: Uso correcto de let, const y var, y manejo de tipos primitivos y objetos.
Funciones y ámbito: Declaración, parámetros, retorno, closures y hoisting.
Manipulación del DOM: Selección de elementos, modificación de contenido y atributos, manejo de eventos y delegación.
Asincronismo en JavaScript: Call Stack, Event Loop, Promesas, async/await, y manejo de errores con try/catch.
Métodos funcionales de arrays: map, filter, reduce, find, y sort para transformar y analizar datos.
Nota: Estos conceptos forman la base para resolver los ejercicios prácticos que siguen. Mantén en mente las mejores prácticas y la claridad en tu código.

Aplicación en el Desarrollo Frontend
En el mundo real, los desarrolladores frontend deben integrar múltiples conceptos para crear interfaces dinámicas, eficientes y robustas. Por ejemplo:

Actualizar dinámicamente el contenido de una página web en respuesta a acciones del usuario.
Gestionar llamadas asincrónicas a APIs para obtener datos y mostrarlos sin bloquear la interfaz.
Manipular estructuras de datos complejas para filtrar y ordenar información mostrada al usuario.
Esta evaluación simula estos escenarios para que puedas demostrar tu capacidad de aplicar lo aprendido en contextos reales y relevantes para la industria.

Instrucciones para la Evaluación Práctica
Para finalizar el curso y poner en práctica todos los conocimientos adquiridos, te proponemos desarrollar una aplicación web JavaScript interactiva, que simula un proceso comercial o profesional.

Objetivos: Desarrollar un simulador interactivo que funcione de igual forma en la cual funciona una aplicación web real, utilizando las diferentes herramientas y técnicas del lenguaje JavaScript aprendida a lo largo del curso, teniendo en cuenta las diferentes limitantes técnicas, reemplazándolas por las herramientas vistas a lo largo del curso para simular el proceso.

Objetivos técnicos:
Manipulación del DOM y eventos:

Tu aplicación JavaScript debe interactuar íntegramente con HTML utilizando DOM y los eventos necesarios.
Arrays:

Todos los arrays de objetos que utilices dentro de tu aplicación deben ser convertidos a un archivo en formato .JSON y accedidos mediante Fetch.
Lógica de tu aplicación:

El simulador interactivo debe completar todo el circuito o proceso de negocio, de acuerdo a la temática del mismo.
Librerías:

Debes incluir al menos una librería JS externa y debes eliminar el uso de herramientas más limitadas del lenguaje, como ser los cuadros de diálogo Prompt, Confirm y Alert, además de eliminar todo rastro de mensajes en la consola
Código claro:

Todo tu código debe ser claro y estar correctamente estructurado para su lectura. Puedes dejar comentarios breves en el código, pero no puedes dejar código en desuso/comentado que complique la lectura y análisis del mismo durante el período de corrección.
Consejo: Aplica buenas prácticas de código, usa nombres descriptivos y comenta tu lógica para facilitar la comprensión.
